# Kids News Reader

### Overview

Web app that allows users to view articles and save comments on the latest news from Dogo News. 
[View Website](https://true-mountie-69018.herokuapp.com/)

### Technologies used
   1. express
   2. express-handlebars
   3. mongoose
   4. cheerio
   5. axios
